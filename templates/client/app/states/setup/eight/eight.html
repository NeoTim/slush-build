Serving Your Staic Files

Now it is time to start plugging things into your application!

Let's go ahead and get a server running to serve up your app.

  • Express
    • One of the top used node/server side frameworks.
    • Express makes everybodies life on the backend much easier, It actually makes it fun!
    • We are going to use what is called express.static.
    • Express.static is a built in method that comes packaged with Express.
    • If you have ever tried serving up static files with node-http, you will know how lovely Express is.
  • Run the following command in the terminal
  • yo ng-guide:server --basic
  • Then run the following command in the terminal
  • node server.js
  • Express static server listening at 127.0.0.1 on port 9000
    Go to localhost:9000
    Or to 127.0.0.1:9000
