Understanding the Client

Checkout the app directory

Notice how each directory has it's own controller and html view.

This is a common best practice. Usually you would put any services, factories, directives and styles and tests that belong to that specific state in the same directory.

  • app.js
    • This is the primary module of the angular application
    • This file must be included in the index.html before any of your angular code
    • This file should also be listed beneath any pre downloaded angular plugins.
  • main
    • The initail main state the app starts out at.
  • steps
    • The steps directory contains all the the steps you are currntly trying to go through.
  • templates
    • This directory contains all the shared html templates
    • For Example: the header, footer, sidebar are inlcuded in the index.html, though the ng-include directive
